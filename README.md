# Gado-Gado FX


Gado-Gado FX is a VST3 plugin based on Juce C++ Framework that contains multiple audio effects like delay, pitch shift, and Filter. 


# Under Development Features!

  - Parameter automation
  - Gain Control
  - Ping-Pong Delay Effect
  - IIR Filter
  - Pitch Shift
